An information panel for Raspberry Pi.

Usage:
- D: night mode
- I: system info
- S: stock info
  - Enter: search
- N: news
  - Up/Down: page up/down
  - Enter: select mode
  - Up/Down (on select mode): news title up/down
  - Enter (on select mode): toggle news image
- M: map
  - Up/Down: select origin/destination entry
  - Tab: toggle mode
  - Enter: search
- B: search on Besy Buy
  - Enter: search
- ESC: back
- Ctrl+Q: quit
